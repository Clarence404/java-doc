# 国内云平台

> 以阿里云为主体，附腾讯云 / 华为云横向对比。国内云在合规、备案、境内网络质量方面有天然优势，是面向国内用户业务的首选。

---

## 一、三大厂商概览

| 对比项 | 阿里云 | 腾讯云 | 华为云 |
|-------|-------|-------|-------|
| 市场份额 | 国内第一（~37%） | 国内第二（~18%） | 国内第三（~17%） |
| 优势 | 服务最全、文档成熟、电商/金融场景深 | 游戏/直播/社交生态、微信生态打通 | 政企/运营商/政务合规、华为硬件生态 |
| 劣势 | 价格偏高 | 部分服务不如阿里云成熟 | 面向企业偏多，个人开发者资源较少 |
| 适合场景 | 互联网、电商、金融、中小企业 | 游戏、直播、泛娱乐 | 政府、运营商、大型企业 |
| 海外节点 | 覆盖较广（东南亚强） | 覆盖一般 | 覆盖一般 |

## 二、阿里云核心服务

### 1、计算

| 服务 | 说明 |
|------|------|
| **ECS** | 弹性云服务器，对应 AWS EC2 |
| **Auto Scaling** | 弹性伸缩，自动增减 ECS 实例 |
| **ACK** | 阿里云容器服务（托管 K8s） |
| **SAE** | Serverless 应用引擎，无需管理节点，直接部署 JAR/WAR |
| **函数计算 FC** | Serverless 函数，对应 AWS Lambda |
| **轻量应用服务器** | 面向个人开发者，价格低，适合建站 / 小项目 |

### 2、存储

| 服务 | 说明 |
|------|------|
| **OSS** | 对象存储，对应 AWS S3，支持图片处理、视频转码 |
| **云盘** | 块存储，挂载到 ECS |
| **NAS** | 网络附加存储，多 ECS 实例共享挂载 |
| **OSS 归档存储** | 冷数据低成本存储，对应 S3 Glacier |

### 3、数据库

| 服务 | 说明 |
|------|------|
| **RDS** | 托管关系型数据库，支持 MySQL / PostgreSQL / SQL Server |
| **PolarDB** | 阿里云自研，兼容 MySQL/PostgreSQL，高性能读写分离 |
| **云数据库 Redis** | 托管 Redis，支持主从 / 集群 / 读写分离 |
| **MongoDB** | 托管 MongoDB |
| **AnalyticDB** | OLAP 数仓，对应 AWS Redshift |
| **Lindorm** | 兼容 HBase/Cassandra 的宽表存储 |
| **TableStore** | Serverless NoSQL，对应 DynamoDB |

### 4、网络

| 服务 | 说明 |
|------|------|
| **VPC** | 私有网络，安全组、子网、路由 |
| **SLB / ALB** | 传统负载均衡 / 应用型负载均衡（7层） |
| **CDN** | 静态资源加速，国内节点覆盖最广 |
| **云解析 DNS** | 域名解析，支持智能线路（联通/电信/移动分线） |
| **高速通道** | 专线接入，对应 Direct Connect |
| **NAT 网关** | 私网 ECS 访问公网出口 |

### 5、消息与中间件

| 服务 | 说明 |
|------|------|
| **RocketMQ** | 阿里自研 MQ，对标 Kafka，事务消息支持好 |
| **Kafka** | 托管 Kafka，适合大数据流处理 |
| **MNS** | 消息通知服务，轻量简单队列 |
| **微服务引擎 MSE** | 托管 Nacos / ZooKeeper / Sentinel，国内微服务首选 |

### 6、监控与运维

| 服务 | 说明 |
|------|------|
| **云监控** | 指标、告警，对应 CloudWatch |
| **日志服务 SLS** | 日志采集、查询、分析，类似 ELK 托管版 |
| **ARMS** | 应用性能监控（APM），支持 Java Agent 接入 |
| **链路追踪** | 分布式追踪，兼容 SkyWalking / Zipkin 协议 |

### 7、安全

| 服务 | 说明 |
|------|------|
| **RAM** | 资源访问管理，对应 AWS IAM |
| **KMS** | 密钥管理 |
| **WAF** | Web 应用防火墙 |
| **DDoS 高防** | Anti-DDoS，T 级防护 |
| **云安全中心** | 主机安全检测（漏洞扫描、入侵检测） |

## 三、阿里云 vs 腾讯云 vs 华为云 核心服务对照

| 服务类型 | 阿里云 | 腾讯云 | 华为云 |
|---------|-------|-------|-------|
| 虚拟机 | ECS | CVM | ECS |
| 托管 K8s | ACK | TKE | CCE |
| Serverless | 函数计算 FC | 云函数 SCF | FunctionGraph |
| 对象存储 | OSS | COS | OBS |
| 关系型数据库 | RDS / PolarDB | TDSQL / CDB | RDS / GaussDB |
| 缓存 | 云数据库 Redis | 云数据库 Redis | 分布式缓存 Redis |
| 消息队列 | RocketMQ / Kafka | CMQ / CKafka | DMS（Kafka/RocketMQ） |
| 负载均衡 | SLB / ALB | CLB | ELB |
| CDN | CDN | CDN | CDN |
| 监控 | 云监控 + ARMS | 云监控 + APM | AOM + APM |
| 微服务注册 | MSE（Nacos） | TSE（Nacos/Consul） | CSE（ServiceComb） |

## 四、Java 后端常用组合

```
典型微服务部署架构（阿里云）：

云解析 DNS
    ↓
CDN（静态资源）+ WAF（安全防护）
    ↓
ALB（负载均衡）
    ↓
ACK（Spring Boot / Spring Cloud 容器集群）
    ↓
RDS MySQL（PolarDB）+ 云数据库 Redis + RocketMQ
    ↓
OSS（文件存储）+ SLS（日志）+ ARMS（APM）
    ↓
MSE（Nacos 服务注册/配置中心）
```

## 五、国内云选型建议

| 场景 | 推荐 | 理由 |
|------|------|------|
| 初创 / 中小互联网 | 阿里云 | 生态最全，文档成熟，迁移成本低 |
| 游戏 / 直播 | 腾讯云 | 游戏 SDK、直播 TRTC 生态完善 |
| 政务 / 金融合规 | 华为云 | 更多合规认证，政企客户支持强 |
| 微信小程序业务 | 腾讯云 | 与微信生态天然打通 |
| 出海 + 国内双栈 | 阿里云 | 国际节点覆盖广，尤其东南亚 |
