# IoT 与工业协议

覆盖 IoT 设备、嵌入式系统、传感器网络和工业自动化场景中的通信协议。

## 官方规范入口

| 协议 | 官方 / 原始规范 |
|------|----------------|
| MQTT | [MQTT 官网](https://mqtt.org/) / [MQTT Specification](https://mqtt.org/mqtt-specification/) |
| CoAP | [RFC 7252](https://www.rfc-editor.org/rfc/rfc7252.html) |
| Modbus | [Modbus Organization](https://www.modbus.org/) |
| OPC UA | [OPC Foundation](https://opcfoundation.org/) / [OPC UA Online Reference](https://reference.opcfoundation.org/) |
| Zigbee | [Connectivity Standards Alliance](https://csa-iot.org/all-solutions/zigbee/) |

- **MQTT（Message Queuing Telemetry Transport）**：轻量级、基于发布订阅的协议，适用于物联网（IoT）应用。
- **CoAP（Constrained Application Protocol）**：专为物联网设计的轻量级协议，类似 HTTP，但更适用于低功耗设备。
- **Modbus**：用于工业自动化的串行通信协议，如 PLC（可编程逻辑控制器）和传感器通信。
- **OPC UA（Open Platform Communications Unified Architecture）**：工业物联网（IIoT）中的常见标准协议。
- **ZigBee**：低功耗无线通信协议，广泛用于智能家居设备。
