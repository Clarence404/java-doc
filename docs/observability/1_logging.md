# 日志体系

> [!warning] 待补充

## 学习目标

- 理解日志在排障、审计、监控和业务分析中的作用
- 区分应用日志、访问日志、错误日志、审计日志
- 掌握结构化日志和日志字段规范

## 大纲

- 日志级别：DEBUG / INFO / WARN / ERROR
- 日志内容：时间、traceId、用户、接口、耗时、错误码
- 日志采集：Filebeat / Fluent Bit / Logstash
- 日志存储与检索：Elasticsearch / Loki / ClickHouse
- 日志规范：脱敏、采样、保留周期、成本控制
