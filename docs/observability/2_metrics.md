# 指标监控

> [!warning] 待补充

## 学习目标

- 理解系统指标、应用指标、业务指标的区别
- 掌握 Prometheus 指标模型和 Grafana 看板设计
- 建立容量、性能、稳定性相关指标意识

## 大纲

- 指标类型：Counter / Gauge / Histogram / Summary
- 基础设施指标：CPU、内存、磁盘、网络
- 应用指标：QPS、RT、错误率、线程池、连接池
- 中间件指标：MySQL、Redis、MQ、JVM
- 业务指标：订单量、支付成功率、转化率
