数据库面试高频题

汇总 MySQL 核心知识的高频面试问题，完整解答见 开发总结-数据库

一、事务与隔离级别

  • MySQL 有哪四种隔离级别？各自能解决什么问题？
  • 脏读、不可重复读、幻读的区别？
  • MySQL 默认隔离级别是什么？为什么能解决幻读？
  • MVCC 的原理是什么？undo log 和 ReadView 如何配合工作？

二、索引

  • B+ 树索引的结构是怎样的？为什么不用 B 树或红黑树？
  • 聚集索引和非聚集索引的区别？什么是回表？
  • 联合索引的最左匹配原则是什么？
  • 索引失效的场景有哪些？（函数、类型转换、like %前缀、OR、不等于…）
  • 如何设计一个合理的索引？覆盖索引是什么？

三、锁

  • MySQL 有哪几种锁？共享锁和排他锁的区别？
  • 行锁、表锁、间隙锁（Gap Lock）、临键锁（Next-Key Lock）的区别？
  • 间隙锁如何防止幻读？
  • 什么情况下行锁会升级为表锁？
  • 高并发下如何安全地修改同一行数据？
    → 详见 MySQL 核心

四、存储引擎

  • InnoDB 和 MyISAM 的区别？（事务、锁粒度、外键、索引结构）
  • InnoDB 为什么推荐使用自增主键？
  • 什么是页（Page）？InnoDB 一个页默认多大？

五、SQL 执行与优化

  • MySQL 的整体执行流程是什么？（连接器 → 分析器 → 优化器 → 执行器）
  • Explain 各字段的含义？type 字段从好到差的顺序是什么？
  • 慢查询如何排查？slow_query_log 如何开启？
  • 几个常见影响 MySQL 性能的案例有哪些？

六、主从与高可用

  • MySQL 主从同步的原理？binlog 有哪几种格式？
  • 主从延迟如何解决？
  • MySQL 有哪几种集群方案？
